## なぜ必要か
配列は作るだけでなく、要素を追加・削除しながら使うことがほとんどだから。

## 何であるか
`push` は末尾に追加、`pop` は末尾を取り出して削除する操作。

## どう使うか
```js
const stack = [];
stack.push("A"); // ["A"]
stack.push("B"); // ["A", "B"]
stack.pop();      // "B" が返り、配列は ["A"]
```
