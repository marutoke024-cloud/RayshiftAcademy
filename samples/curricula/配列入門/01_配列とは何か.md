## なぜ必要か
複数の値をひとつの変数でまとめて扱いたいことが多いから。

## なぜそう設計されているか
連続したインデックスでアクセスできるようにすることで、要素の取り出しが高速になる。

## 何であるか
配列は「順序を持つ値の集まり」。各要素には 0 から始まる番号（インデックス）が振られる。

## どう動くか
`arr[0]` のように添字でアクセスすると、その位置の要素が返る。

## どう使うか
```js
const fruits = ["りんご", "みかん", "ぶどう"];
console.log(fruits[1]); // みかん
```

先輩、配列の番号が 0 から始まるのは最初は戸惑いますが、慣れるとむしろ自然に感じますよ！
